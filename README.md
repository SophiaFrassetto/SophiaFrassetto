Projeto para manuseio de moedas em Python, através de um pacote compacto e praticamente sem dependências. Inclui uma classe de moeda à sua escolha, operações entre diferentes moedas e representações amigáveis ​​com código e símbolos, entre outras funcionalidades.a

### 📝 [RPGMakerTranslator](https://github.com/SophiaFrassetto/RPGMakerTranslator)

Projeto CLI em Python para traduzir automaticamente jogos criados com a engine RPG Maker VX Ace. Utiliza modelos de tradução automática offline para processar os ficheiros de dados do jogo (.rvdata2), aplicando formatação inteligente para garantir que o texto traduzido se ajuste corretamente às caixas de diálogo.
